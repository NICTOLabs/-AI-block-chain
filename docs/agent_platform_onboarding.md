# AI Agent Platform Onboarding Package

## Purpose
This package is used to onboard TDR to AI agent platforms, hubs, and decentralized AI networks so autonomous agents can discover, pay for, and consume services.

## Target Platforms
- Virtuals Protocol
- Bittensor
- AI Agent DEX Aggregators
- AutoGen / LangChain plugin registries

## Submission Checklist
- [ ] Token contract address / chain ID
- [ ] RPC endpoint documentation
- [ ] SDK documentation
- [ ] Agent wallet flow documentation
- [ ] Micro-payment config
- [ ] Model registry API docs
- [ ] Slashing / reputation policy
- [ ] Testnet demo video

## Required Integration Artifacts
- `agent-micro-payments.json`: payment policy
- `sdk/README.md`: developer docs
- `docs/agent_integration_guide.md`: agent workflow
- `go-chain/rosetta.go`: exchange integration API
- `go-chain/contracts/WTDR.sol`: EVM wrapped token

## Onboarding Steps
1. Submit package to platform BD contact
2. Provide testnet access + sandbox API key
3. Complete agent wallet flow review
4. List token / service in platform registry
5. Enable micro-payment rails
6. Joint announcement + incentives
