## Target Launchpads

## Application Checklist
- [ ] Audited smart contracts / core protocol
- [ ] Public tokenomics + vesting schedule
- [ ] Legal opinion / utility token classification
- [ ] Working testnet or mainnet deployment
- [ ] Community metrics (Discord, Twitter, GitHub)
- [ ] Liquidity plan (DEX pool + lock duration)
- [ ] Market maker agreement
- [ ] Post-launch roadmap with milestones

## Required Docs
- Whitepaper
- Token economics
- Team bios + verification
- Legal/compliance memo
- Security audit report
- Liquidity lock proof

## Proposed IDO Terms
- **Token price:** $0.02 - $0.05 per TDR
- **Raise target:** $2M - $5M
- **Token allocation:** 2% - 5% of total supply
- **Vesting:** 20% at TGE, then 80% over 12 months
- **Market maker:** 12-month lockup
- **DEX liquidity:** 12-month lockup minimum
